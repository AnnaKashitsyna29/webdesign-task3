Об игре

Pixel Jumper - классический 2D платформер. Управляйте персонажем, собирайте монеты, избегайте препятствий и достигайте финиша!

Особенности

  • 3 уровня с разной сложностью
  • Подвижные платформы
  • Система сохранения рекордов
  • Эффекты частиц
  • Адаптивное управление

О разработке

Игра полностью разработана на чистом JavaScript с использованием Canvas API. Создано с помощью AI (Cursor/DeepSeek).
